お望みのテキストを自然な音声で読み上げます

Chrome と Safari で利用可能な Web アプリ。 ダウンロードやインストールは必要ありません。 面倒なログインも必要ありません。 テキストをペーストし、「再生」をクリックするだけ。

ポッドキャストのテキスト

TTSReader は、他のことをしている間にもテキストを聞きたい、そんな忙しい人のための便利なツールです。
例えば ​​、Wikipediaの記事、仕事や個人的な勉強のための文書を聴くことができます（ここにコピー＆ペーストするだけです）。

モバイルデータのトラフィックや料金を大幅に抑えます

通常のポッドキャストアプリでは、長時間のオーディオファイルのサイズが大きいためにデータ量も増大します。 TTSReader なら、手元の携帯システムに内蔵されている音声エンジンを使いテキストを読み上げます。 そのため、ほぼゼロに近いデータ消費量でポッドキャストのテキストを長時間聴くことができます。 このソフトウェアは、運転中や外出先でも興味のあるコンテンツを聞きたい、そんな方に最適です。


子供の教育目的

子供たちは良い話を聴くことが大好きで、それは成長のためにも大切なことです。 ママやパパが物語を読み聞かせることに勝るものはありません。 しかし、TTSReader を使えば、ママやパパが読み聞かせに加え、さらに多くの物語を聞かせてあげることができます。 お試しください、きっと子供たちは気に入るはずです！

使い方、その他

使い方

アプリケーションのテキストボックスに、読み上げたいテキストをペーストします。
「再生」をクリックします。 これだけです。 あとは TTSReader に任せて、楽しんで聴いてください。

一時停止と再生 - 中央の「再生 / 一時停止」ボタンをクリックすると、一時停止や読み上げの再開ができます。
最初から再生 - 最初から読み上げ直すには、「巻き戻し」ボタンをクリックします。
言語の設定、読み上げ速度 - 言語と読み上げ速度は、必要に応じて変更できます。 次の言語をサポートしています：英国英語、米国英語、スペイン語、フランス語、イタリア語、ドイツ語、日本語、韓国語、中国語。


互換性

TTSReader はChrome や Safari 上で実行させるため、プラットフォームを問いません。 デスクトップ、Mac、ラップトップ、Chromebook、スマートフォンで使うことができます。
